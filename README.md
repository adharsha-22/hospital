 This project performs basic data analysis on hospital patient records. It helps to understand patient demographics, common diseases, and billing patterns using Python.
*File Name:* hospital.csv
*Columns in Dataset:*
1. *Patient-ID* - Unique identification number for each patient
2. *Name* - Full name of the patient  
3. *Age* - Age of the patient in years
4. *Gender* - Gender of the patient (Male/Female)
5. *Disease* - Disease or medical condition diagnosed
6. *Admission-Date* - Date when the patient was admitted
7. *Bill amount* - Total hospital bill amount in Rupees
1. Find the *total number of patients* in the dataset
2. Identify which *disease is most common* among patients
3. Analyze the *gender ratio* of admitted patients
4. Calculate the *average bill amount* across all patients
- *Disease Distribution Analysis* - Bar chart showing count of each disease
- *Gender Distribution* - Pie chart showing male vs female percentage
- *Billing Analysis* - Average bill amount and patient-wise billing- *Programming Language:* Python 3
- *Libraries:* Pandas for data manipulation, Matplotlib & Seaborn for data visualization
- *Platform:* Can be executed on Google Colab or Local Python Environment
1. Download the hospital.csv file from this repository
2. Upload the file to Google Colab or place it in your local project folder
3. Run the analysis script to generate charts and statistics
After running the analysis, you will get:
- Text output showing total patients and average bill
- Visual charts showing disease count and gender distribution
This is a beginner-level data analysis project suitable for learning Pandas and data visualization basics.
